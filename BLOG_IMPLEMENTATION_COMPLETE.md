## 技術規格

### 數據庫設計

**blogPosts 表**：
- 23 列，包括三語言標題、內容、摘要
- 支援分類、標籤、關鍵詞
- 包含發佈狀態、發佈日期、瀏覽計數
- 自動時間戳（createdAt、updatedAt）

**caseStudies 表**：
- 27 列，包括三語言標題、摘要、挑戰、解決方案、成果
- 支援項目分類、技術棧、客戶名稱
- 包含完成日期、特色圖像 URL
- 自動時間戳

### 後端 API

所有 API 端點均使用 tRPC 框架，支援：
- 類型安全的查詢和變更
- 自動序列化（SuperJSON）
- 錯誤處理和驗證

### 前端頁面

使用 React 19 + Tailwind CSS 4 + shadcn/ui 組件：
- 響應式設計（桌面、平板、手機）
- 暗色/亮色主題支持
- 無障礙設計（WCAG 2.1）

### SEO 優化

- 動態 Meta 標籤（title、description、keywords）
- Open Graph 標籤（og:title、og:description、og:image）
- Twitter Card 標籤
- JSON-LD 結構化數據（BlogPosting、BreadcrumbList）
- Canonical 標籤
- 內部連結策略
