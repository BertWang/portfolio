# SEO 監測儀表板與 AI 引用追蹤指南

## 概述

本文檔說明如何建立 SEO 監測儀表板，追蹤王純瑋作品集網站在搜尋引擎和 AI 系統中的表現。

---

## 第一部分：核心監測指標

### 1. 搜尋引擎排名

**追蹤指標**：
- 目標關鍵詞排名（台南網頁設計、PHP 開發等）
- 長尾關鍵詞排名
- 排名變化趨勢

**工具**：
- Google Search Console（免費）
- Ahrefs（付費）
- Semrush（付費）
- SE Ranking（中等價格）

**目標**：
- 主關鍵詞：前 3 位
- 長尾關鍵詞：前 10 位

### 2. 有機流量

**追蹤指標**：
- 月度有機訪問量
- 有機流量增長率
- 按頁面的流量分佈

**工具**：
- Google Analytics 4（免費）
- Hotjar（付費）

**目標**：
- 第 4 週：+10-15%
- 第 8 週：+25-35%
- 第 12 週：+40-60%

### 3. AI 引用率

**追蹤指標**：
- 在 ChatGPT 中被引用的次數
- 在 Perplexity 中被引用的次數
- 在 Claude 中被引用的次數
- 在 DeepSeek 中被引用的次數

**工具**：
- Cairrot（推薦，支持 5 大 LLM）
- Peec AI（支持 Looker Studio 集成）
- Monitoro（情緒分析）
- ZipTie.dev（GEO 診斷）

**目標**：
- 第 4 週：+20-30%
- 第 8 週：+40-60%
- 第 12 週：+50-80%

### 4. 用戶行為

**追蹤指標**：
- 平均停留時間
- 跳出率
- 頁面瀏覽數
- 轉化率

**工具**：
- Google Analytics 4
- Hotjar（熱力圖）
- Microsoft Clarity

**目標**：
- 平均停留時間：> 2 分鐘
- 跳出率：< 50%
- 轉化率：> 5%

## 第二部分：Google Search Console 設置

### 1. 基本設置

1. 訪問 [Google Search Console](https://search.google.com/search-console)
2. 添加網站屬性：`https://bertfolio-vncqhm2r.manus.space`
3. 驗證所有權（使用 DNS 記錄或 HTML 標籤）

### 2. 關鍵詞監測

**設置搜尋分析報告**：

1. 進入「效能」報告
2. 添加過濾器：
   - 查詢：台南網頁設計、PHP 開發、品牌設計等
   - 頁面：/blog、/blog/:id

3. 導出數據到 Google Sheets 進行進一步分析

**關鍵詞清單**：

| 關鍵詞 | 搜尋量 | 目標排名 | 當前排名 |
|------|------|--------|--------|
| 台南網頁設計 | 高 | 1-3 | TBD |
| PHP 開發 | 中 | 1-5 | TBD |
| 品牌設計 | 中 | 1-5 | TBD |
| 古蹟數位化 | 低 | 1-3 | TBD |
| 文化保存設計 | 低 | 1-3 | TBD |

### 3. 索引覆蓋率

**監測項目**：
- 有效頁面數
- 排除的頁面
- 錯誤頁面

**目標**：
- 所有博客頁面都被索引
- 沒有 4xx 或 5xx 錯誤

### 4. 核心網絡指標（Core Web Vitals）

**監測指標**：
- LCP（最大內容繪製）：< 2.5 秒
- FID（首次輸入延遲）：< 100 毫秒
- CLS（累積布局偏移）：< 0.1

**改進方法**：
- 優化圖片大小
- 使用 CDN
- 減少 JavaScript 體積
- 啟用緩存

## 第三部分：Google Analytics 4 設置

### 1. 事件追蹤

**設置自定義事件**：

```javascript
// 追蹤內部連結點擊
gtag('event', 'internal_link_click', {
  'link_url': url,
  'link_text': text,
  'page_path': window.location.pathname
});

// 追蹤相關文章點擊
gtag('event', 'related_article_click', {
  'article_id': articleId,
  'article_title': articleTitle,
  'source_article_id': sourceArticleId
});

// 追蹤 FAQ 點擊
gtag('event', 'faq_click', {
  'faq_question': question,
  'article_id': articleId
});
```

### 2. 轉化追蹤

**定義轉化**：
- 諮詢表單提交
- 電話撥號
- 郵件訂閱
- 下載資源

**設置轉化**：
1. 進入「轉化」設置
2. 創建新的轉化事件
3. 設置轉化價值（可選）

### 3. 報告和儀表板

**創建自定義報告**：

1. **SEO 效能報告**
   - 按頁面的有機流量
   - 按頁面的轉化率
   - 按關鍵詞的流量

2. **內容效能報告**
   - 按文章的訪問量
   - 按文章的停留時間
   - 按文章的轉化率

3. **用戶行為報告**
   - 新用戶 vs 回訪用戶
   - 用戶來源
   - 用戶設備

## 第七部分：優化建議流程

### 1. 數據收集

- 每週收集上述所有指標
- 記錄到 Google Sheets
- 生成圖表和趨勢線

### 2. 分析

- 識別性能最好的內容
- 識別性能最差的內容
- 分析排名變化的原因
- 分析流量變化的原因

### 3. 優化

根據分析結果，進行以下優化：

**高性能內容**：
- 增加內部連結指向
- 更新和擴展內容
- 在社群媒體上推廣

**低性能內容**：
- 分析原因（排名低、內容質量差等）
- 重新優化或重寫
- 添加更多內部連結
- 改進 SEO 元數據

### 4. 報告

每月生成一份報告，包括：
- 關鍵指標摘要
- 排名變化
- 流量變化
- AI 引用變化
- 優化建議
