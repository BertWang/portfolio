## 設置步驟

### 1. 創建 Google Sheet

1. 訪問 [Google Sheets](https://sheets.google.com)
2. 點擊「新建」→「試算表」
3. 命名為「王純瑋作品集 - SEO 監測儀表板」
4. 創建以上 6 個表單

### 2. 連接 Google Analytics 4

```
在 Google Sheets 中：
1. 菜單 → 擴充功能 → 取得附加元件
2. 搜尋「Google Analytics」
3. 安裝「Google Analytics 4 for Sheets」
4. 授權連接
5. 在表單中使用 =GA4() 函數導入數據
```

**範例公式**：

```
=GA4(
  "properties/YOUR_PROPERTY_ID",
  "2026-06-09",
  "2026-06-15",
  "activeUsers,sessions,screenPageViews",
  "date"
)
```

### 3. 連接 Google Search Console

```
在 Google Sheets 中：
1. 菜單 → 擴充功能 → 取得附加元件
2. 搜尋「Search Console」
3. 安裝「Search Console for Sheets」
4. 授權連接
5. 在表單中使用 =SEARCHCONSOLE() 函數導入數據
```

**範例公式**：

```
=SEARCHCONSOLE(
  "https://bertfolio-vncqhm2r.manus.space",
  "query",
  "2026-06-09",
  "2026-06-15",
  "clicks,impressions,ctr,position"
)
```

### 4. 手動導入 Cairrot 數據

由於 Cairrot 沒有直接的 Sheets 集成，需要手動導入：

1. 訪問 Cairrot 儀表板
2. 導出 CSV 報告
3. 複製數據到 Google Sheets

### 5. 創建圖表和儀表板

在 Google Sheets 中：

1. 選擇數據
2. 菜單 → 插入 → 圖表
3. 選擇圖表類型
4. 自定義標題、軸標籤等

**推薦圖表**：

- **有機流量趨勢**：折線圖（週次 vs 有機流量）
- **排名進度**：折線圖（週次 vs 平均排名）
- **AI 引用增長**：柱狀圖（週次 vs AI 引用）
- **關鍵詞排名分佈**：圓餅圖（排名位置分佈）
- **文章性能對比**：橫條圖（文章 vs 流量）

## 關鍵指標解釋

### 有機流量

**定義**：來自搜尋引擎的訪問數

**目標**：
- 第 1-4 週：+10-15%
- 第 5-8 週：+25-35%
- 第 9-12 週：+40-60%

**提升方法**：
- 發佈高質量內容
- 優化內部連結
- 改進 CTR

### 排名位置

**定義**：目標關鍵詞在搜尋結果中的平均位置

**目標**：
- 第 1-4 週：-2 到 +2 位
- 第 5-8 週：+2 到 +5 位
- 第 9-12 週：+5 到 +8 位

**提升方法**：
- 增加內容深度
- 改進頁面 UX
- 增加反向連結

### AI 引用率

**定義**：在 AI 系統中被引用的次數

**目標**：
- 第 1-4 週：+20-30%
- 第 5-8 週：+40-60%
- 第 9-12 週：+50-80%

**提升方法**：
- 優化結構化數據
- 編寫 llms.txt
- 提高內容質量

## 常見問題

**Q1：如何確保數據準確性？**
A：定期驗證 Google Sheets 中的數據與原始來源（GA4、GSC、Cairrot）的一致性。

**Q2：應該多久更新一次數據？**
A：每週一次。定期更新可以幫助及時發現趨勢和問題。

**Q3：如何處理數據異常？**
A：記錄異常事件（如網站宕機、算法更新）並在分析時考慮這些因素。

**Q4：如何與團隊共享儀表板？**
A：在 Google Sheets 中點擊「共享」，添加團隊成員的郵箱並設置權限。

**Q5：如何自動化數據導入？**
A：使用 Google Apps Script 編寫自動化腳本，定期從 API 導入數據。
